# Server Side
Server side for Home Security

## Installation
1. Install [NodeJS][1]
2. Install dependencies
    - Run [install.bat][2]

[1]: https://nodejs.org/en/
[2]: ./install.bat


# Usage
To run server open runserver.bat

---

Install dependencies
`$npm install`

To run server manually
`$npm run start:watch`

## Note
- The server runs in PORT 8080
